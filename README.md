# 会议管理系统
系统采用SSM的技术架构，纯注解整合，实现了用户管理、会议管理、参会管理等核心功能模块。通过JSP 技术构建前端界面，实现了良好的用户交互体验。项目整体运行稳定，基本满足了用户的日常会议管理需求。

很水的项目，适合做学习比较差的人，只保个及格的人的的期末课设
